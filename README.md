# zxh-ui

> 基于vue开发的常用组件

## Install

``` bash
npm install zxh-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import ZUI from 'zxh-ui'
import 'zxh-ui/lib/styles/index.css'

Vue.use(ZUI)
```

For more information, please refer to [Quick Start](https://zxh742755443.github.io/zxh-ui/) in our documentation.
