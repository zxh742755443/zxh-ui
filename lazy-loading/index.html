    # LazyLoading 懒加载

    图片懒加载。

    # 基础用法

    在图片标签上添加 v-lazy-loading，并赋值相应的图片地址，可实现图片懒加载。

    # 渐变色背景

    当网速慢的时候，图片还没加载完之前，用户会看到一段空白的时间，在这段空白时间，可以使用渐变色背景。v-lazy-loading 值添加 background，可仔细观察 demo 最后一张图片
